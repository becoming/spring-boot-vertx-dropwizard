== Setup

=== Build and package:
```bash
mvn clean package
```

=== Run:

==== via Maven
```bash
mvn clean package spring-boot:run
```

*OR*

==== via Java
```bash
java -jar spring-boot-0.0.1-SNAPSHOT.jar
```
