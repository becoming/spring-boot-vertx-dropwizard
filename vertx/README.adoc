= Vertx

This application was generated using http://start.vertx.io

== Setup

=== Build and package:
```
mvn clean package
```

=== Run:

==== via Maven
```
mvn clean package exec:java -DskipTests=true
```

*OR*

==== via Java
```bash
java -jar vertx-0.0.1-SNAPSHOT-fat.jar --run
```
